# Claude Code Portable

> [Claude Code](https://claude.ai/code) 的便携打包版本——解压即用，无需安装。

本 README 在每次发布后自动重新生成。请查看 [Releases](../../releases) 页面获取下载链接。

## 快速开始

从 [最新发布](../../releases/latest) 页面下载对应平台的归档文件，解压后即可运行：

```bash
# Linux / macOS
tar xzf claude-portable-linux-x64-vX.Y.Z.tar.gz
./claude-portable-linux-x64-vX.Y.Z/claude --version
```

平台专属说明和代理配置请参阅发布页面。
