# Claude Code Portable

> Portable packaging of [Claude Code](https://claude.ai/code) — extract and run, no installation required.

This README is automatically regenerated after each release. Please check the [Releases](../../releases) page for download links.

## Quick Start

Download the archive for your platform from the [latest release](../../releases/latest), extract it, and run:

```bash
# Linux / macOS
tar xzf claude-portable-linux-x64-vX.Y.Z.tar.gz
./claude-portable-linux-x64-vX.Y.Z/claude --version
```

See the release page for platform-specific instructions and proxy configuration.
